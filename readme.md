Main exe/main.py needs to be in the same folder as a file called "qr.png" or the program **will** crash.

## How to run
To run the python script
1. Open a terminal
2. 
```bash
python3 main.py
```

To run the exe. Open the exe.

## How to compile
Run compile.cmd
